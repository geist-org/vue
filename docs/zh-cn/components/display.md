</ex-code>

</ex-code>

| ---------- | ---------- | ---- |  -------------- | ------ |

| **caption** | 说明文字 | `string` | `template` | - | - |
| **shadow** | 阴影效果 | `boolean` | - | `false` |

</ex-footer>
