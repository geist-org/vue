| Attribute  | Description      | Type      | Accepted values | Default |
| ---------- | ---------------- | --------- | --------------- | ------- |
| **src**    | image url        | `string`  | -               | -       |
| **width**  | image width attr | `string`  | -               | `auto`  |
| **shadow** | shadow style     | `boolean` | -               | `false` |
